## data:appinfo 모듈
- 앱 자체 정보에 대한 데이터 접근 관련 기능들이 정의되어 있습니다.

### data:appinfo 모듈 내 패키지 구조
- datasource : repository가 데이터를 가져오는 대상인 데이터 저장소를 구현한 클래스
- repository : domain 레이어에서 정의한 repository 인퍼테이스를 구현한 클래스
